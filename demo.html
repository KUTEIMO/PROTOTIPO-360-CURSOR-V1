Demo guiada del MVP accesible

Utiliza esta guía rápida para presentar el recorrido virtual y verificar los controles clave.

Preparación

  1. Ejecuta npm install la primera vez.
  2. Inicia el servidor con npm run start y abre http://localhost:8080/index.html.
  3. Activa un lector de pantalla (NVDA, VoiceOver o Narrador) para validar mensajes aria-live.

Pasos sugeridos para la demostración

  1. Usa los enlaces iniciales “Saltar al menú” y “Saltar al primer hotspot” para mostrar los skip-links.
  2. Abre el menú lateral con el botón hamburguesa, navega con Tab y selecciona “Biblioteca Planta Principal”.
  3. Recorre los hotspots con Tab y activa uno con Enter. Valida que se abra el panel, se anuncie vía aria-live y que la audiodescripción pueda reproducirse.
  4. Activa el modo de alto contraste y aumenta el tamaño del texto desde la barra de accesibilidad.
  5. Presiona el botón de ayuda para revisar la lista de atajos y ciérralo con Escape.
  6. Navega con las flechas de teclado y los botones anterior/siguiente para mostrar la navegación progresiva.

Evidencias a capturar

  • Captura de la escena con el panel de hotspot abierto y subtítulos visibles.
  • Resultado del comando npm run audit mostrando los archivos generados en reports/.
  • Salida de npm test verificando el enfoque de hotspots.
